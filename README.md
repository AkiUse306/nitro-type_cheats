## Installation

### Method 1: Browser Console (Easiest)
1. Open Nitro Type in your browser
2. Press `F12` or `Ctrl+Shift+I` to open Developer Tools
3. Go to the **Console** tab
4. Copy and paste the desired cheat code from the `/cheats` folder
5. Press Enter to execute

### Method 2: Bookmarklet
1. Create a new bookmark in your browser
2. Paste the code from a cheat file as the URL
3. Click the bookmark while on Nitro Type to activate the cheat

### Method 3: Userscript (Tampermonkey/Greasemonkey)
1. Install [Tampermonkey](https://www.tampermonkey.net/) extension
2. Create a new script
3. Copy the content from `/userscripts` folder
4. Save and enable the script
