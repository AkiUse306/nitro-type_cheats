## 🛠️ Troubleshooting

### "Cheat not working"
- Make sure you're on the correct page (nitrotype.com)
- Check console for error messages
- Try refreshing the page
- Make sure console is cleared (no previous errors)

### "Console disappeared"
- Press `F12` or `Ctrl+Shift+I` again to reopen it

### "Money didn't add"
- Refresh the page after pasting the code
- Check your balance manually (might need to wait a moment)

### "Auto-typer typing too fast/slow"
- Edit the `speed` value in the code
- Slower = higher number (try 100-200)
- Faster = lower number (try 30-50)
